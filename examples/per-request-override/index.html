Per-Request Override

全局配置 successCode: [0]、retry: { count: 2 }，在单个请求中可覆盖这些配置

1. successCode 覆盖

全局 successCode: [0]，旧接口返回 code: 200 表示成功。不覆盖会报 Business Error，覆盖后正常解析

2. retry 覆盖

全局 retry: { count: 2, delay: 500 }。可以给单个请求禁用 retry，或者用不同的 retry 配置

3. responseTransform 覆盖

全局默认解包返回 data，单个请求可获取原始 AxiosResponse 或自定义转换
